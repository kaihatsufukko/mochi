# のどかな縁側

寺尾ワカメの個人サイト「のどかな縁側」のソースコードです。

## サイト概要

日記やニュースの発信をメインとした個人サイトです。コメント欄や拍手ボタンを通じて訪問者との交流を図ります。

## 技術スタック

- **フロントエンド**: React 19 + TypeScript + Tailwind CSS 4
- **UIコンポーネント**: shadcn/ui + Radix UI
- **アニメーション**: Framer Motion
- **ルーティング**: Wouter
- **ビルドツール**: Vite

## デザインコンセプト

「昭和の縁側」をテーマにした、ノスタルジックで温かみのあるデザインです。

- 畳の淡い緑と木の茶色を基調とした配色
- 障子の生成り色の背景
- 朱色のアクセント（拍手ボタンなど）
- Zen Maru Gothic（丸ゴシック）の見出しで親しみやすさを演出
- 本文18px、行間1.9で高齢者にも読みやすいレイアウト

## ページ構成

| ページ | パス | 説明 |
|--------|------|------|
| トップページ | `/` | 最新記事一覧、日記・ニュースセクション |
| 日記一覧 | `/diary` | 日記記事の一覧（タグフィルター付き） |
| ニュース一覧 | `/news` | ニュース記事の一覧（タグフィルター付き） |
| 記事詳細 | `/article/:id` | 記事本文、拍手ボタン、コメント欄 |

## 主な機能

- **拍手ボタン**: ワンクリックでリアクション（花びらアニメーション付き）
- **コメント機能**: 記事ごとにコメントを投稿可能
- **タグフィルター**: タグで記事を絞り込み
- **レスポンシブ対応**: スマホファースト設計
- **アクセシビリティ**: 大きな文字、高コントラスト、タップしやすいボタン

## 開発

```bash
# 依存パッケージのインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build
```
